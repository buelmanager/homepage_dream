# AZUR YACHTS - Luxury Yacht Charter Website

## Industry
Luxury Yacht Charter

## Language
English

## Style
- **Theme**: Dark luxury maritime
- **Primary Background**: Deep Navy (#0A1628)
- **Accent Color**: Gold (#D4A853)
- **Text Color**: Warm Light (#F0EDE6)
- **Fonts**: Playfair Display (headings) + Inter (body)
- **Aesthetic**: Premium, elegant, nautical with subtle gold accents

## Sections
1. **Preloader** - Compass-style rotating animation with gold sweep line and brand reveal
2. **Navbar** - Fixed navigation with scroll-triggered background blur and mobile hamburger menu
3. **Hero** - Full-screen diagonal split with ocean background, luxury yacht image, wave animation divider
4. **Marquee** - Scrolling yacht brand names (Azimut, Ferretti, Sunseeker, Benetti, Princess, Lurssen, Feadship, Riviera)
5. **Charter Packages** - Three-tier pricing cards (Day Cruise $2,500 / Weekend Escape $8,500 / Week Voyage $25,000)
6. **Before/After Slider** - Draggable comparison between standard and luxury cabin experience
7. **Process Timeline** - 5-step journey (Inquiry, Route Planning, Provisioning, Boarding, Voyage)
8. **Gallery** - 8-image grid with blue gradient overlay on hover, featuring yacht and ocean photography
9. **FAQ** - 5-question accordion covering charter details, booking, customization, and weather policies
10. **Contact/Booking** - Dual-column layout with company info + booking form (Name, Email, Phone, Yacht Preference, Dates, Special Requests)
11. **Footer** - Full footer with company description, services links, fleet links, information links, and social icons

## Features
- GSAP ScrollTrigger animations for all sections
- Parallax hero background effect
- CSS wave animation divider between hero and content
- Compass-style preloader with rotating needle
- Interactive before/after image comparison slider (mouse + touch)
- FAQ accordion with smooth GSAP height transitions
- Gold top border hover effect on package cards
- Blue gradient overlay on gallery images on hover
- Wave sweep hover effect on package cards
- Mobile-responsive design with hamburger menu
- Smooth scroll navigation
- Form submission feedback animation
- Fully self-contained single HTML file (no external CSS/JS dependencies beyond CDN)
