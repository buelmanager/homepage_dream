# Clone Plan — Cordage & Co. (rope-workshop)

## Brand Concept
**Cordage & Co.** is a fictional Victorian rope-making workshop on the Devon coast, England. Established 1861. Produces bespoke natural-fibre rope for maritime, architectural, and decorative purposes.

## Design Language
- **Palette**: Warm sienna and linen (#1e1a14 bg, #b87333 sienna, #f2ead8 linen)
- **Typography**: Libre Baskerville (display) + Cormorant Garamond (body) + Inter (UI)
- **Hero Type**: Type C — Diagonal Split (55% image / 45% text, CSS clip-path polygon)
- **Signature elements**: CSS repeating-linear-gradient woven rope texture, Victorian craft aesthetic

## Pages
1. `index.html` — Preloader + 13 mandatory sections
2. `about.html` — Workshop history, master craftspeople, heritage timeline
3. `collection.html` — Commission formats (horizontal rows) + natural fibre sourcing grid
4. `process.html` — 5 alternating image/text craft stages (Victorian tools)
5. `contact.html` — Workshop location + bespoke commission form

## Differentiation from Existing Templates
- Victorian maritime craft concept entirely new to library
- Diagonal clip-path hero with woven CSS texture is unique visual treatment
- "Natural fibre sourcing" as luxury narrative is novel
- Warm sienna palette (#b87333) distinct from existing warm-toned templates
