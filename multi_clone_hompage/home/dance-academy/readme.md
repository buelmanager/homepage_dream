# Dance Academy — CADENCE Contemporary Dance & Movement Studio

## Overview
A premium single-page website template for a contemporary dance academy. Dark, expressive, and dynamic — built for dance studios, movement academies, and performing arts schools.

## Cloned From
`multi_clone_hompage/home/pilates-studio/index.html` (ALIGN — Pilates & Barre Studio)

## Design Decisions

### Color Palette
- **Background**: `#0D0A1A` (deep indigo-black)
- **Surface**: `#1A1530` (dark purple)
- **Accent**: `#A78BFA` (soft violet)
- **Text**: `#F5F3FF` (lavender white)
- **Muted**: `#8B83A3` (muted lavender)

### Typography
- **Headings**: Playfair Display (elegant serif with italics)
- **Body**: Inter (clean modern sans-serif)

### Sections
1. **Preloader** — Stage spotlight sweep animation
2. **Hero** — Dancer silhouette with dramatic spotlight gradients
3. **Marquee** — Dance styles scrolling: Contemporary, Hip Hop, Ballet, Jazz, K-Pop, Lyrical
4. **Classes** — 6 dance style cards with level badges (Beginner/Intermediate/Advanced)
5. **Schedule** — Weekly timetable with color-coded class types
6. **Instructors** — 4 dance teacher cards with photos, specialties, and credentials
7. **Gallery** — Studio & performance photos with inner glow hover
8. **FAQ** — Dance academy specific questions
9. **Contact** — Registration & inquiry with trial class pricing
10. **Footer** — Social links, class list, info, support

### Key Differentiators from Source
- Spotlight preloader (vs silk ribbon flow)
- Violet/indigo palette (vs pink/navy)
- Playfair Display + Inter (vs DM Serif Display + DM Sans)
- Dance-specific content: choreography, styles, performances
- Floating light particles on hero (vs silk ribbons)
- All content in English (source had Korean mixed)
- IntersectionObserver animations (no ScrollTrigger for reveals)

## Tech Stack
- HTML5 / CSS3
- GSAP 3.12 (timeline, fromTo animations)
- IntersectionObserver (scroll reveal, iframe compatible)
- Google Fonts (Playfair Display + Inter)
- Unsplash images (free commercial use)
