# Verdant Veil — Orchid Greenhouse Luxury Landing Page

## Concept

**Verdant Veil** is a luxury orchid greenhouse brand that positions itself as an exclusive sanctuary for rare botanical treasures. The brand embodies patience, dedication, and the art of cultivation.

### Brand Identity

- **Brand Name**: Verdant Veil
- **Tagline**: "Where Patience Meets Petals"
- **Brand Voice**: Refined, contemplative, nature-forward

### Target Audience

- High-net-worth individuals seeking rare botanical specimens
- Orchid collectors and enthusiasts
- Corporate clients seeking luxury gifts
- Wedding and event planners

## Section Structure

1. **Preloader** — Brand mark animation + logo reveal
2. **Scroll Indicator** — Left-fixed navigation with progress
3. **Navbar** — Fixed header with blur effect on scroll
4. **Hero** — Cinematic fullscreen with Ken Burns effect
5. **Stats Strip** — 4-column key metrics with counter animation
6. **Philosophy** — 3-column grid explaining core values
7. **Collection** — 2x2 grid with hover effects
8. **Atelier** — 2-column sanctuary introduction
9. **Process** — 5-step vertical timeline
10. **Heritage** — 4-column milestone timeline
11. **Press** — Swiper carousel with testimonials
12. **Commission Form** — 2-column contact section
13. **Footer** — 4-column navigation

---

## Animations

- **Preloader**: SVG draw + brand fade-in
- **Hero**: Ken Burns background, split-text content reveal
- **Stats**: Counter animation triggered on scroll
- **Cards**: Stagger reveal with translateY
- **Process Timeline**: Step-by-step fade-in
- **Press Carousel**: Swiper with autoplay
- **Forms**: Input focus animations
