## Reference Sites Analyzed

### 1. William Henry (williamhenry.com)
- **Elements borrowed**: Dark background aesthetic, dramatic product photography style, material callout format
- **Changes made**: Different color palette (ours is warmer, more forge-focused), unique commission application form, no pricing display
- **Reasoning**: Adapted luxury knife presentation but with our own scarcity positioning

### 2. Jonas Blade (jonasblade.com)
- **Elements borrowed**: Process-focused storytelling, forge-first imagery
- **Changes made**: Our 5-stage timeline is more detailed, we include heritage section, different visual hierarchy
- **Reasoning**: Adopted craft-first tone but structured differently

### 3. Flett Forge (flettforge.com)
- **Elements borrowed**: Place-rooted narrative concept
- **Changes made**: Our location is fictional (Seki), we use coordinates instead of address, different historical dates
- **Reasoning**: Mythologized location approach inspired but completely fictional

### 4. Canton Cutler (via Smorgasbord Studio)
- **Elements borrowed**: Editorial typography contrast concept
- **Changes made**: Different font pairing (Cormorant + DM Sans vs whatever they use), different color scheme
- **Reasoning**: High-contrast typography principle adapted

## Original Elements

1. **Brand Identity**: IGNIS LAMA is fictional - Latin-inspired name created specifically for this project
2. **Tagline**: "Forged in fire. Finished in silence." - Original
3. **Scarcity Model**: 12 blades/year concept - unique positioning
4. **Commission Gate**: Form as application, not order - inverts power dynamic
5. **Animated Hex Grid Hero**: Custom CSS animation, not found in any reference
6. **Scroll Indicator**: Custom dot + line + label + progress fill design
7. **All Copy**: All text content is original, written for this brand
8. **Section Structure**: While following luxury landing conventions, specific ordering and content is original
