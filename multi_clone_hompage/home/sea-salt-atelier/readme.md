# Salinae — Sea Salt Atelier Luxury Landing Page

**Brand:** Salinae Atelier (fictional)
**Tagline:** Where the Ocean Crystallises Into Art
**Industry:** Artisan Sea Salt / Coastal Luxury
**Hero Type:** Type D — Portrait + Stats (large numbers on left, portrait on right)

## Brand Identity

Salinae is a fictional ultra-premium artisan sea salt atelier founded by a fictional master paludier named Margaux Delval. The brand harvests from three ocean sites — Atlantic Brittany, Mediterranean Sicily, and Pacific Okinawa — using a proprietary twelve-pond solar evaporation gradient system that retains all 84 natural mineral trace elements.

**Color Palette:**
- Background: `#151C22` (deep ocean night)
- Surface: `#1D2830` (dark slate blue)
- Tide accent: `#4A8FA8` (coastal teal-blue)
- Tide light: `#7DBDD6` (clear shallow water)
- Salt text: `#E8EDF0` (sea salt white)
- Muted: `#6A8496`

**Typography:**
- Display: Playfair Display (classical, editorial luxury)
- Headings: Raleway (light, coastal modernism)
- Body: Inter (precision, clarity)
