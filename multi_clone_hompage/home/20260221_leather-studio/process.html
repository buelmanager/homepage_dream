Process

Each VERAK piece passes through five essential stages. No shortcuts. Only time-honored techniques.

01 / SELECTION

Hide Selection

We inspect every hide personally. Only full-grain vegetable-tanned leather with natural grain and character makes the cut.

02 / CUTTING

Hand Cutting

Each piece is cut by hand using traditional patterns. We account for grain direction and natural markings.

03 / EDGING

Edge Preparation

Edges are beveled, sanded, and burnished. Multiple rounds of treatment create smooth, durable edges.

04 / STITCHING

Hand Stitching

Using saddle stitch technique. If one thread breaks, the other holds. Each stitch is pulled tight by hand.

05 / FINISHING

Final Polish

Edges are sealed, hardware is attached, and the piece is conditioned. The maker's mark is pressed in.
