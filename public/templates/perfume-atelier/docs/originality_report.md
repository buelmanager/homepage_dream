# Originality Report — SÈVE Parfums d'Exception

## Reference Sites Studied
1. Byredo (byredo.com) — dark minimal fragrance brand
2. Xerjoff (xerjoff.com) — ultra-luxury niche perfumery
3. Memo Paris (memoparis.com) — bespoke fragrance positioning
4. Roja Parfums (rojaparfums.com) — heritage luxury house aesthetic

## Inspired Elements & Changes Made

| Reference Element | Inspired By | Changes Made |
|---|---|---|
| Dark background palette | Byredo, Xerjoff | Darker bg (#0E0B0A vs industry avg #151515), amber gold vs silver/white accents |
| Bottom-line form inputs | Common luxury forms | Used with transparent bg + gold focus state |
| Fixed left scroll indicator | Multiple luxury sites | Added progress fill + animated line vs simple dot |
| Press carousel | Standard implementation | Swiper 11 with 3 architectural quotes, not just logos |
| 2-col atelier section | Memo Paris split layout | Added GSAP parallax scrub on image, gold frame decoration |
