# Shizuku — Matcha Cafe Luxury Landing Page

## Concept
A fictional Kyoto-inspired matcha sanctuary where ceremony is the offering, not the product. Positioned against the "wellness green" trope with dark luxury aesthetics borrowed from fine dining and perfume brands.

**Brand:** Shizuku (雫 = Dewdrop)
**Tagline:** Still. Green. Perfect.
**Positioning:** Ultra-premium, ritual-focused, single-origin, ceremony-as-experience

## Typography
- **Serif:** Playfair Display — editorial contemplation, western serif with Japanese restraint
- **Body:** DM Sans — clean, modern café precision
- **JP Decorative:** Noto Serif JP — floating kanji elements (静, 茶) in hero

## Section Architecture
1. **Preloader** — circular arc SVG draw + Shizuku brand name
2. **Scroll Indicator** — left-fixed dot system
3. **Navbar** — transparent → forest blur on scroll
4. **Hero** — Ken Burns dark café + floating kanji + mousemove parallax
5. **Stats Strip** — 4 ceremony metrics with counter animation
6. **Philosophy** — 3-col CSS grid (STILL word + pillars + tea image)
7. **Seasonal Menu** — 3-col offset grid with hover detail overlay
8. **Ceremony Atelier** — 2-col split + parallax image
9. **Single Origin Timeline** — 5 Japanese terroirs, spine draw
10. **Heritage** — 5 generational milestones (1852–2024)
11. **Press** — Swiper fade carousel
12. **Reserve Form** — 2-col ceremony booking
13. **Footer** — 4-col minimal
