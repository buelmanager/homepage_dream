# VELVET SCREEN - Private Cinema Experience

## Industry
Boutique Private Cinema / Entertainment Venue

## Language
English

## Style
- **Theme**: Dark, cinematic, premium luxury
- **Background**: Deep black (#0D0D0D)
- **Accent**: Crimson red (#C41E3A)
- **Surface**: Dark charcoal (#1A1A1A)
- **Text**: Warm light (#F5F0EB)
- **Muted Text**: Warm gray (#8A8078)
- **Fonts**: Bebas Neue (headings) + DM Sans (body)

## Sections
1. **Preloader** - Film countdown animation (3, 2, 1) with circular SVG progress, crimson sweep line, brand reveal
2. **Navbar** - Fixed navigation with scroll effect, mobile hamburger menu, "Book Screening" CTA
3. **Hero** - Full-screen diagonal split layout with cinema interior background, spotlight vignette effect, "Your Story, Your Screen" headline
4. **Marquee** - Scrolling feature tags (Dolby Atmos, 4K Laser, IMAX Quality, Private Suites, Gourmet Dining, VIP Lounge, Film Library, Premiere Events)
5. **Packages** - 3 pricing tiers: Solo Screen ($150), Date Night ($350, featured), Grand Premiere ($1,200)
6. **Before/After** - Draggable comparison slider: standard theater vs. VIP private cinema
7. **Process** - 5-step timeline: Browse, Select, Customize, Arrive, Experience
8. **Gallery** - 8 cinema/theater images from Unsplash with film-strip perforated edge styling
9. **FAQ** - 5 accordion items covering capacity, food/beverage, movie selection, booking/cancellation, technical specifications
10. **Contact** - Screening booking form with Name, Email, Phone, Package select, Preferred Date, Special Requests; venue info with operating hours
11. **Footer** - Company info, Experiences links, Venue links, Information links, social icons

## Features
- **GSAP Animations** - ScrollTrigger-powered reveal animations for all sections
- **Film Countdown Preloader** - Animated 3-2-1 countdown with SVG circle progress
- **Before/After Slider** - Mouse and touch-enabled drag comparison
- **FAQ Accordion** - Animated open/close with GSAP, single-item-open behavior
- **Mobile Responsive** - Full responsive design with hamburger menu for mobile
- **Parallax Hero** - Background parallax scroll effect
- **Film Strip Hover** - Horizontal lines pattern hover effect on package cards (replaces chrome sweep)
- **Film Strip Gallery** - Perforated edge borders on gallery items simulating 35mm film strip
- **Section Dividers** - Thin red lines with centered diamond shape
- **Spotlight Vignette** - Radial gradient overlay on hero section for cinematic depth
- **Red Curtain Gradients** - Package cards feature curtain-inspired gradient at top edge
- **Smooth Scrolling** - Anchor links with offset for fixed navbar
- **Form Submission** - Animated success feedback on form submit
- **Unsplash Images** - All images sourced from Unsplash (cinema, theater, movie screen, luxury seats, popcorn, film)
