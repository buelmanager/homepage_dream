# SUMMIT LODGE — Alpine Luxury Chalet Resort

## Industry
Luxury Alpine Hospitality / Mountain Chalet Resort

## Language
English

## Style
Dark forest theme with warm gold accents, evoking a luxurious mountain lodge atmosphere. Dark forest background (#0F1A12) paired with warm gold (#C9A96E) creates an intimate, high-end alpine feel. Typography uses Cormorant Garamond (serif) for elegant headings and Outfit (sans-serif) for clean body text.

## Color Palette
| Role       | Color     | Hex       |
|------------|-----------|-----------|
| Background | Dark Forest | `#0F1A12` |
| Accent     | Warm Gold   | `#C9A96E` |
| Surface    | Deep Green  | `#1A2A1E` |
| Text       | Warm White  | `#F2EDE4` |
| Muted Text | Sage Green  | `#8B9A80` |

## Sections
1. **Preloader** — Mountain peak line-art SVG animation with falling snow CSS particles, golden glow sweep loader, brand reveal
2. **Navigation** — Fixed navbar with scroll-triggered background blur, mobile hamburger menu with slide-in panel
3. **Hero** — Full-viewport diagonal split (mountain landscape / luxury chalet interior), parallax mountain silhouette layers (3 SVG layers at different scroll speeds), "Where Mountains Meet Luxury" headline
4. **Marquee** — Scrolling alpine activities ticker (Skiing, Snowboarding, Spa & Wellness, Gourmet Dining, Helicopter Tours, Ice Climbing, Wine Cellar, Stargazing)
5. **Accommodation Packages** — 3-tier pricing cards (Alpine Suite $450/night, Premium Chalet $1,200/night featured, Grand Estate $3,500/night) with warm glow sweep hover
6. **Before/After Slider** — Summer vs winter chalet view comparison with draggable handle (mouse + touch)
7. **Process Timeline** — 5-step guest journey (Discover, Reserve, Arrive, Explore, Relax) with numbered step circles and connecting line
8. **Gallery** — 8-image grid with first image spanning 2x2, warm-tone filter overlay, zoom-on-hover
9. **FAQ Accordion** — 5 questions about seasons, amenities, ski access, dining, and airport transfers with GSAP-animated expand/collapse
10. **Contact/Reservation** — Split layout with resort info + reservation form (Name, Email, Phone, Room Type select, Check-in/Check-out dates, Special Requests)
11. **Footer** — 4-column layout with brand description, experiences, accommodations, and information links

## Features
- **GSAP + ScrollTrigger** animations for section reveals, parallax, and interactive elements
- **CSS-only falling snow** particle effect in preloader (20 snowflakes with staggered timing)
- **Parallax mountain silhouettes** — 3 SVG mountain layers moving at different scroll speeds in the hero
- **Mountain ridge SVG dividers** between sections for organic visual flow
- **Before/After comparison slider** with drag support (mouse and touch)
- **Warm glow sweep** hover effect on package cards (golden gradient instead of chrome)
- **Wood-grain texture border** on package card hover
- **Warm-tone filter overlay** on gallery images
- **FAQ accordion** with GSAP height animation and icon rotation
- **Fully responsive** — mobile menu, stacked layouts for tablet/phone
- **Single-file architecture** — all CSS, HTML, and JS in one self-contained file
- **No external dependencies** beyond Google Fonts and GSAP CDN
- **Unsplash images** for all photography (alpine, chalet, mountain, skiing, luxury, dining, spa)

## Fonts
- **Headings**: Cormorant Garamond (Google Fonts) — elegant serif for luxury feel
- **Body**: Outfit (Google Fonts) — modern, clean sans-serif
