# Luxury Portrait Studio Website Template

## Overview
Premium luxury portrait photography studio website with elegant, high-end design featuring champagne gold accents and sophisticated dark theme.

## Industry
Luxury Portrait Photography

## Business Type
High-end portrait photography studio specializing in:
- Executive/Corporate portraits
- Family heirloom portraits
- Celebrity/Media portraits
- Fashion editorial photography
- Children & Baby portraits
- Fine art portraits
- Pet portraits

## Language
English

## Style
- **Theme**: Luxury, Elegant, Sophisticated
- **Color Palette**:
  - Primary: #0A0A0A (Deep Black)
  - Accent: #D4AF37 (Champagne Gold)
  - Accent Light: #E8C872 (Light Gold)
  - Background: #050505 (Rich Black)
  - Surface: #141414 (Elevated Dark)
  - Text: #F5F0E8 (Warm Cream)
  - Text Muted: #8A8578 (Warm Gray)
- **Typography**:
  - Headings: Playfair Display (Serif)
  - Body: Cormorant Garamond (Serif)
- **Visual Effects**:
  - Golden iris preloader animation
  - Subtle grain overlay
  - Smooth reveal animations
  - Lightbox gallery
  - Hover effects with gold accents

## Sections
1. **Preloader** - Golden iris shutter animation
2. **Navbar** - Fixed navigation with transparency
3. **Hero** - Dramatic portrait background with stat cards
4. **Client Marquee** - Luxury brand logos
5. **Services** - 8 portrait service categories
6. **Gallery** - Masonry grid with lightbox
7. **Artists** - Master photographer showcase
8. **Testimonials** - Client stories
9. **FAQ** - Common questions
10. **Contact/Booking** - Reservation form
11. **Footer** - Links and contact info

## Features
- Fully responsive design
- Smooth scroll animations
- Interactive lightbox gallery
- FAQ accordion
- Mobile-friendly navigation
- Contact form with service selection
