# Clone Plan — Lumière Noire (vintage-cinema)

## Brand Concept
**Lumière Noire** is a fictional private cinema atelier on the Rive Gauche, Paris. Founded 1924 by Édouard Lumière-Beaumont. Presents rare and archival films in intimate screenings of 24 guests maximum.

## Design Language
- **Palette**: Film-noir dark (#1a1510 bg, #c9a84c gold, #f0e8d8 cream)
- **Typography**: Playfair Display (display) + Cormorant Garamond (body) + Inter (UI)
- **Hero Type**: Type A — Cinematic Fullscreen Ken Burns
- **Signature elements**: SVG film grain overlay, filmstrip CSS borders, Ken Burns 18s animation cycle

## Pages
1. `index.html` — Preloader + 13 mandatory sections
2. `about.html` — Story grid, team custodians, core values
3. `collection.html` — Screening programme archive (6 series)
4. `process.html` — Curation process, projection specs, film archive
5. `contact.html` — Patron enquiry + salon location details

## Differentiation from Existing Templates
- Film-noir colour palette distinct from all existing dark templates
- Ken Burns animation with filmstrip CSS border decoration is unique
- "Patron cinema" concept not present in any existing template
- Curation-focused copy (films as sacred objects) differentiates from event venues
