# Luxury Spa — SERENITÉ Luxury Spa & Wellness Retreat

## Overview
A serene, luxurious single-page website template for a premium spa and wellness center. Dark, organic, and tranquil — built for luxury spas, wellness retreats, and holistic therapy centers.

## Cloned From
`multi_clone_hompage/home/interior-design/index.html` (ATELIER BLANC — Interior Design Studio)

## Design Decisions

### Color Palette
- **Background**: `#060808` (deep forest black)
- **Surface**: `#0F1412` (dark moss)
- **Border**: `#1E2A25` (forest border)
- **Accent**: `#8DB48E` (sage/eucalyptus green)
- **Accent Dark**: `#6B9A6C` (deep sage)
- **Accent Dim**: `#5A8A6A` (muted eucalyptus)
- **Text**: `#F0F5F1` (misty white)
- **Text Muted**: `#7A8E80` (sage gray)

### Typography
- **Headings**: Cormorant (elegant serif)
- **Body**: Lato (clean modern sans-serif)

### Sections
1. **Preloader** — Lotus flower SVG bloom animation (petals unfold sequentially)
2. **Navbar** — Brand logo with sub-text, smooth scroll navigation
3. **Hero** — Split layout: left text with service categories + right spa image with floating detail and "PURE BLISS" badge
4. **Intro** — "The Art of Wellness" with 4 feature cards (Holistic Therapy, Natural Botanicals, Expert Therapists, Serene Spaces)
5. **Spa Experiences** — 3 project cards (Thermal Suite large, Treatment Rooms, Meditation Garden)
6. **Wellness Journey** — 5-step timeline process (Welcome Ritual → Wellness Assessment → Personalized Treatment → Deep Relaxation → Aftercare)
7. **Natural Ingredients** — 4 swatch cards (Lavender, Eucalyptus Oil, Sea Minerals, Volcanic Clay)
8. **Testimonials** — Guest reviews with Swiper carousel
9. **Contact** — Booking form with spa-specific fields
10. **Footer** — Brand info, navigation, services, social links

### Key Differentiators from Source
- Lotus bloom preloader (vs floor plan line drawing)
- Sage/eucalyptus green on forest-black (vs silver-slate on ultra-dark)
- Cormorant + Lato (vs Sora + Inter)
- Spa/wellness content: treatments, botanicals, relaxation
- Spa experience showcase (vs interior project cards)
- Wellness journey timeline (vs design process)
- Natural ingredient swatches (vs construction materials)
- Grid pattern preserved (adds zen/structured feel)

## Tech Stack
- HTML5 / CSS3
- Vanilla JavaScript (preloader, navbar, mobile menu)
- Swiper 11 (testimonials carousel)
- Google Fonts (Cormorant + Lato)
- Unsplash images (free commercial use)
- Grid pattern background
- Custom scrollbar styling
