# HomeDream — 프로젝트 계획서

> AI 시대 바이브코더를 위한 홈페이지 템플릿 마켓플레이스
> 생성일: 2026-02-15

---

## 1. 프로젝트 개요

### 비전
바이브코딩 시대에 고퀄리티 홈페이지를 크레딧으로 저렴하게 구매할 수 있는 마켓플레이스

### 레퍼런스
- **monet.design** — UI/UX 철학, 그리드 레이아웃, 디바이스 프리뷰 시스템
- **shadcn/ui** — 컴포넌트 아키텍처, copy-paste 철학
- **v0 by Vercel** — 크레딧 기반 모델, AI 네이티브 워크플로우

### 차별화
1. **크레딧 기반 구매** (monet.design은 무료/오픈소스)
2. **완성형 HTML 템플릿** (React 컴포넌트가 아닌 독립 실행 HTML)
3. **30개 실제 템플릿** (카페, 법률사무소, 피트니스 등 다양한 업종)
4. **관리자 패널** (폴더에 HTML 넣으면 자동 등록)
5. **한국어/영어** 다국어 지원

## 2. 기술 스택

| 영역 | 기술 | 이유 |
|------|------|------|
| 프레임워크 | Next.js 15 (App Router) | SSR/SSG, SEO, monet.design과 동일 |
| 언어 | TypeScript | 타입 안전성 |
| 스타일링 | Tailwind CSS v4 | 빠른 개발, monet.design과 동일 |
| UI 라이브러리 | shadcn/ui + Radix UI | 접근성, 커스터마이징 |
| 데이터베이스 | PostgreSQL (Supabase) | 확장성, 무료 티어 |
| ORM | Prisma 7 | 타입 안전 쿼리 |
| 인증 | NextAuth.js v5 | 이메일/비밀번호 + OAuth |
| 애니메이션 | Framer Motion | 부드러운 UI 전환 |
| 아이콘 | Lucide React | 깔끔한 아이콘 |
| 패키지 매니저 | npm | 범용성 |

## 3. 데이터베이스 스키마

### 핵심 모델
- **User**: 사용자 (이메일, 크레딧, 역할)
- **Template**: 템플릿 (슬러그, 제목, 가격, 카테고리, 태그, 조회수)
- **Section**: 섹션 (템플릿의 개별 파트)
- **Purchase**: 구매 기록
- **Favorite**: 찜 목록
- **Bookmark**: 북마크
- **CreditTransaction**: 크레딧 거래 로그

### 시드 데이터
- 30개 실제 HTML 템플릿 (multi_clone_hompage/home에서 복사)
- 180개 섹션
- 관리자 + 데모 사용자

## 4. 페이지 구조

```
/ .......................... 홈페이지 (히어로 + 카테고리별 그리드)
/templates ................. 전체 템플릿 목록 (필터/검색)
/templates/[slug] .......... 템플릿 상세 (디바이스 프리뷰 + 구매)
/c/[slug] .................. 개별 섹션/컴포넌트 페이지
/category/[name] ........... 카테고리별 목록
/leaderboard ............... 인기 랭킹 (조회수/좋아요/저장)
/profile ................... 사용자 프로필 (구매/찜/북마크)
/admin ..................... 관리자 대시보드
/admin/upload .............. 템플릿 업로드
/signin .................... 로그인
/signup .................... 회원가입
```

## 5. 핵심 기능

### 5.1 템플릿 그리드 (monet.design 스타일)
- 4열 반응형 그리드
- 카드 호버: 어두운 오버레이 + "Preview" / "View Details" 버튼
- 태그, 섹션 수, 언어 배지 표시

### 5.2 디바이스 프리뷰
- Desktop (1280px) / Tablet (768px) / Mobile (375px) 전환
- iframe 기반 실제 HTML 렌더링
- 새로고침, 전체 프리뷰, 원본 방문 버튼

### 5.3 크레딧 시스템
- 신규 가입시 100 크레딧 지급
- 템플릿 구매: 13-24 크레딧
- 섹션 구매: 5-8 크레딧
- 구매 후 HTML 소스 다운로드 가능

### 5.4 관리자 패널
- 템플릿 CRUD
- public/templates/ 폴더 스캔으로 자동 등록
- HTML 파일 업로드

### 5.5 리더보드
- 일간/주간/전체 기간 필터
- 조회수/좋아요/저장 기준 정렬
- 카테고리별 필터
